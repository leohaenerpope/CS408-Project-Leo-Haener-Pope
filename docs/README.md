# Full-Stack Web Application
This repository will contain a full-stack web application built with Python, Django, and SQLite.

## Technology Stack
- Backend technology stack
    - Web Server: [nginx](https://nginx.org/)?
    - Backend Runtime: [Python](https://www.python.org/)
    - Backend Framework: [Django](https://www.djangoproject.com/)
    - Database: [SQLite](https://sqlite.org/)
- Frontend technology stack
    - Templates: [Django integrated templates](https://docs.djangoproject.com/en/6.0/topics/templates/)
    - UX/UI: [Bootstrap 5](https://pypi.org/project/django-bootstrap5/)
- Testing Frameworks:
    - End-to-End Testing: [Python pytest](https://docs.pytest.org/en/stable/)
